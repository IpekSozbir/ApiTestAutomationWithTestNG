# ApiTestAutomationWithTestNG

Bu repo, REST API test otomasyonu çalışmalarımı içeren bir projedir.
Projede Java, Rest-Assured ve TestNG kullanılarak uçtan uca API test senaryoları geliştirilmiştir.

## :rocket: Proje Amacı

Bu projenin amacı;

- REST servislerinin otomatik test edilmesi
- Farklı HTTP metodlarının (GET, POST, PUT, DELETE) test edilmesi
- Response doğrulamalarının yapılması
- POJO ve JSON yapıları ile veri modelleme
- TestNG ile organize edilmiş test senaryoları oluşturulması

üzerine pratik ve gerçekçi test otomasyon örnekleri sunmaktır.

---

## :hammer_and_wrench: Kullanılan Teknolojiler

- Java 17
- Rest-Assured
- TestNG
- Gson
- JSON Library
- Maven

---

## :package: Bağımlılıklar (pom.xml)

Projede kullanılan temel kütüphaneler:

- Rest-Assured → API testleri için
- TestNG → Test yönetimi ve koşumu için
- Gson → JSON verilerini serileştirme/deserileştirme
- org.json → JSON işlemleri
- JUnit → Yardımcı test anotasyonları (test scope)
